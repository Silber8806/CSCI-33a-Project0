SCSS Example

We will use SASS to compile an SCSS file. The SCSS File will be present on this screen. The compiled output will be presented below that. We will use the compiled CSS within this webpage to highlight some basic things within the code.

SCSS Code 

            
                $term_color :  black;
                $term_text_color : white;
                $term_margin : 25px;
                $term_width: 500px;

                pre {
                    background-color:$term_color;
                    margin:$term_margin;
                    width:$term_width;

                    code {
                    color: $term_text_color;
                    }
                }

                %comp {
                    span.var {
                        color: red;
                    }

                    span.prop {
                        color: lightblue;
                    }

                    span.class {
                        color: orange;
                    }

                    span.instance {
                        color: magenta;
                    }

                    span.primitive {
                        color: blue;
                    }
                }

                .python {
                @extend %comp;
                    color: green;
                }

                .scss {
                @extend %comp;
                    color: white;
                }

Compile CSS Code 

             
                 pre {
                  background-color: black;
                  margin: 25px;
                  width: 500px; }
                    pre code {
                     color: white; }

                .python span.var, .scss span.var {
                  color: red; }
                .python span.prop, .scss span.prop {
                  color: blue; }
                .python span.class, .scss span.class {
                  color: purple; }
                .python span.instance, .scss span.instance {
                  color: magenta; }

                .python  {
                  color: green; }

                .scss {
                  color: white; }
