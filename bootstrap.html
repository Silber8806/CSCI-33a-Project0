Bootstrap4!

CSCI-S33a includes a section on Bootstrap 4. The below are examples of bootstrap 4 that are slight modications of the examples on the main tutorial page. I show how to create 2-columns, staggered 3 columns and 3-columns with Bootstrap 4 cards attached.

Two-Column row

You can add two divs within a row with col-sm and it will automatically split the container into 2. If there are 3 divs each div would take up 1/3rd of a container.
